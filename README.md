Todo Server

Це простий сервер на Node.js з використанням Express, який обробляє маршрути для користувачів та статей.

Встановлення

Клонувати репозиторій:

git clone https://github.com/AnnaBatmaeva/home-work60.git

Встановити залежності:

npm install


Запустити сервер:

node app.js


Сервер за замовчуванням слухає порт 3000.
